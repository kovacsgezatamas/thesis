# Badge component - Thesis UI

<br />

## Installation

```sh
$ yarn add @thesis-ui/badge
$ npm install @thesis-ui/badge
```

<br />

## Usage

```jsx
import Badge from '@thesis-ui/badge';

<AnyContainer>
  <Badge
    isInner
    position="bottomRight"
  >
    10
  </Badge>
</AnyContainer>
```
